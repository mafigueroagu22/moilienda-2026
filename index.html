Persistencia de Datos

Tus datos se guardan automáticamente
PERSISTENTE
Este dashboard usa localStorage de tu navegador Chrome. Esto significa que:

✓ Los datos NO se borran al cerrar la pestaña o el navegador.
✓ Los datos se mantienen mientras no limpies el caché de Chrome.
✓ Cada actualización que hagas se suma a los datos existentes.

⚠ Los datos son locales a este navegador y computador. Si abres el archivo en otro PC, partirá con los datos iniciales.

Cómo Actualizar la Información

Opción 1 — Pídeme que lo actualice (recomendado)
FÁCIL
Sube tu archivo Data_Cemento.xlsx actualizado a esta conversación y dime:

"Actualiza el dashboard de optimización con estos nuevos datos"

Yo proceso el Excel automáticamente y te genero una nueva versión del HTML con los datos actualizados. Al abrirlo, los datos nuevos se guardarán en tu navegador.
Opción 2 — Agregar datos manualmente desde el dashboard
DIRECTO
Usa el formulario de abajo para agregar un nuevo mes. Los datos se guardan automáticamente en tu navegador y todos los gráficos se actualizan al instante.
