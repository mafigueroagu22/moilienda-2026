# Optimización Molienda 2026 — Planta Coronel

Dashboard de monitoreo del informe ejecutivo de optimización de molienda de cemento.

**Polpaico Soluciones** · Jefatura de Producción

## Características

- KPIs en tiempo real: Rendimiento, kWh/t, Factor Clínker
- Seguimiento Fck Meta vs Actual (CP400 y CESP)
- 5 alternativas de optimización fundamentadas
- Matriz de riesgos y cumplimiento NCh148
- Plan de acción en 5 fases
- **Datos persistentes** (localStorage)
- Formulario integrado para agregar datos nuevos

## Acceso

Abrir en: `https://TU-USUARIO.github.io/molienda-2026/`

## Actualización de datos

Los datos se guardan en el navegador del usuario. Para agregar nuevos meses, usar la pestaña "Cómo Actualizar" dentro del dashboard.
